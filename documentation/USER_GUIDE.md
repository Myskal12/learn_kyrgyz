# User Guide

## 1. What The App Is For

I built Learn Kyrgyz to support short daily practice sessions in Kyrgyz through:

- flashcards
- quizzes
- sentence building
- category roadmap screens
- review for difficult words

## 2. How I Start

1. I open the app.
2. I complete the welcome flow.
3. I choose a daily goal.
4. I continue as a guest or sign in.
5. I use the main CTA on Home to begin the next study action.

## 3. Main Screens

### Home

I use Home to see the best next action:

- continue the last lesson
- review due words
- follow the daily goal

### Practice

I use Practice to choose a training mode directly:

- flashcards
- quick quiz
- sentence builder

### Categories

I use Categories as the topic roadmap.

There I can:

- choose a category
- view topic progress
- open a lesson or continue a lesson

### Progress

I use Progress to check:

- streak
- mastered words
- weak words
- review due
- next milestone
- sync state

### Profile and Settings

I use these screens for:

- theme
- daily goal
- study direction
- account and sync settings

## 4. Typical Study Session

My usual study cycle is:

1. learn or review words in flashcards
2. check recall in quiz
3. reinforce usage in sentence builder
4. revisit mistakes through review

## 5. Study Direction

I change the study direction in Settings.

That setting affects the exercises, but it does not stay visible on every exercise screen.

## 6. Offline Use

The app partially supports offline use:

- settings and progress are stored locally
- content can load from local cache
- sync can continue later when the network returns

If there is no internet connection, cloud-dependent features may be limited.

## 7. If Something Goes Wrong

- If content does not load, I check the network and try again.
- If sync fails, the app keeps local progress and retries later.
- If Google Sign-In fails on Android, I check the Firebase setup and SHA keys.
